# XXDP Test names

XXDP tests use letter codes to identify what the test is meant for.

### 1st letter code

|     |     |
| --- | --- |
| A = 11/05, 11/15, 11/20  <br>B = 11/40  <br>C = 11/45  <br>D = GT40, general UNIBUS  <br>E = 11/70  <br>F = 11/34  <br>G =11/04  <br>H = used for system files of run time environment  <br>J = 11/23,11/24  <br>K = 11/44 | M = MNC/11  <br>N = System Industry 9400 controller,11/21  <br>P = PDT-11, Plessey diagnostics  <br>Q = 11/60  <br>R = LPA-11  <br>V = 11/03 LSI11  <br>X = X11 module  <br>Z = unspecified/ any platform |
